# ComptaPlus

Application web de gestion de factures développée en HTML, CSS, JavaScript et jQuery.

## Fonctionnalités

- Gestion des clients
- Gestion des produits et services
- Création de factures
- Calcul automatique HT, TVA et TTC
- Export des factures au format PDF
- Sauvegarde des données avec LocalStorage

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- jQuery
- LocalStorage
- jsPDF
- jsPDF-AutoTable

## Utilisation

1. Télécharger le projet.
2. Ouvrir le fichier `index.html` dans un navigateur.
3. Utiliser l’application pour créer et exporter des factures.
